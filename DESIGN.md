# Design System

## Current: Surgical Dark with Sky Blue

Based on chief-of-staff site aesthetic.

### Colors

| Token | Value | Usage |
|-------|-------|-------|
| `--bg-primary` | `#0a0a0a` | Page background |
| `--bg-card` | `#141414` | Cards, sections |
| `--accent` | `#60A5FA` | Links, highlights, CTAs |
| `--border` | `#222222` | Borders |
| `--text-primary` | `#f0f0f0` | Headings |
| `--text-secondary` | `#888888` | Body text |
| `--text-muted` | `#555555` | Labels, dates |

### Typography

- **Body:** Inter
- **Code/Mono:** JetBrains Mono
- **Headers:** Light weight (300-400), tight letter-spacing
- **Labels:** Uppercase, wide letter-spacing (0.05-0.1em)

### Aesthetic

- Clinical, precision, techwear-inspired
- Zero shadows, only borders
- Minimal border-radius (2px)
- Sharp, surgical feel
